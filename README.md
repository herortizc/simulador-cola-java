# Simulador de Cola de Atención al Cliente

Proyecto del curso Estructuras de Datos.
Implementación de Cola (FIFO) y Pila (LIFO) utilizando nodos y referencias.
No se utilizan colecciones de Java.

## Compilación

javac src/*.java

## Ejecución

java -cp src Simulador

## Estructura

src/
  Nodo.java
  Cola.java
  Pila.java
  Simulador.java

data/
  clientes.txt
